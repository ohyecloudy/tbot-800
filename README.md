# tbot-800

twitter bot

## Installation

    $ git clone https://github.com/ohyecloudy/tbot-800.git ~/project/tbot-800
	$ cd ~/project/tbot-800
	$ lein deps
	$ cp _config.clj config.clj

modify config.clj

	$ lein run
